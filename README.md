# Student File Handler

A centralized platform designed to efficiently manage and organize all student files. The goal is to minimize confusion and make file sharing and access seamless for students and instructors.

---

## 🚀 Features

* **Centralized File Management:** Upload and manage all files in a single, organized place.
* **Role-Based Access Control:** Secure file access with different roles (e.g., Admin, user);
* **Simplified Structure:** A clean, intuitive design to reduce confusion and improve usability.
* **Flexible File Storage:** Integrates with various cloud storage solutions like Backblaze B2, with support for local storage.

---

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js, MongoDB
* **Frontend:** React.js (under development)
* **Cloud Storage:** Backblaze B2 (or other providers)

---

## 📆 Installation

Follow these steps to get the backend server up and running on your local machine. The frontend setup instructions will be added soon.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Krishna240517/Student-File-Handler.git](https://github.com/Krishna240517/Student-File-Handler.git)
    cd Student-File-Handler
    ```

2.  **Install backend dependencies:**
    ```bash
    cd backend
    npm install
    ```

3.  **Start the backend server:**
    ```bash
    npm run start
    ```
4. **Start the frontend server:**
   ```bash
   npm run dev
   ```
